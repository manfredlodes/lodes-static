# lodes-static

## Projekt

Migration der WordPress-Website **lodes.ch** zu einer modernen statischen Website.

## Ziele

- Keine Datenbank
- Kein WordPress
- HTML5
- CSS3
- Modernes JavaScript (ES6+)
- JSON als Datenquelle
- Responsive Design
- Hohe Performance
- Barrierefreiheit
- SEO
- DSGVO-konform
- Wartungsarm
- Erweiterbar
